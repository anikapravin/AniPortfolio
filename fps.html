Stylised FPS UI

📌 This project aims to revamp the weapons inventory user interface of a mobile shooter game to embrace a cyberpunk grunge aesthetic. This aesthetic combines futuristic technology with a worn, gritty, and dystopian atmosphere, creating an immersive experience for players.

🔧 The themed interface is maintained throughout, with distressed textures, graffiti-like markings, and rusty accents. The user can navigate through the inventory and explore various actions related to each weapon.
